# Tang Tang Balance Research Codex

`탕탕특공대(Survivor!.io)`의 공개 자료 기반 게임 구성기획 및 밸런스 리서치를 저장한 폴더입니다.

## 구성

- `01_tangtang_public_design_balance_summary_ko.md`
  - 공개 자료를 바탕으로 정리한 한국어 통합 요약
- `02_source_catalog.md`
  - 조사에 사용한 주요 공식/준공식/커뮤니티 자료 링크와 활용 메모
- `03_public_data_gaps_and_followups.md`
  - 공개 자료만으로 확인되지 않는 항목과 후속 조사 포인트

## 주의

- 이 폴더의 내용은 공개 자료를 종합한 리서치입니다.
- 내부 개발 문서, 내부 밸런스 테이블, 비공개 수치 데이터는 포함하지 않습니다.
- 일부 시스템 해석은 출처 기반 종합 분석이며, 해당 파일 안에 별도로 구분해 두었습니다.
