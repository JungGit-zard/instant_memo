# 공개 자료 공백과 후속 조사 포인트

## 현재 공개 자료만으로 비어 있는 항목

- 챕터별 몬스터 등장 테이블
- 웨이브 타이밍과 보스 패턴의 정량 비교표
- 무기/장비/캐릭터/테크 파츠 최신 수치 표준화 목록
- 버전별 밸런스 패치 전수 비교
- 한국 서비스 전용 운영/이벤트 차등 여부
- 과금 패키지 가격/가치의 시계열 변화
- 실제 상위권 메타 전환 시점 기록

## 후속 조사 추천 순서

1. 공식 패치노트/업데이트 로그 수집
2. 무기/EVO/장비/캐릭터/테크 파츠 표준화 테이블 작성
3. 모드별 메타 분리 분석
4. F2P와 과금 유저의 성장 격차 분석
5. 한국 서비스 관점 BM/라이브옵스 구조 정리
