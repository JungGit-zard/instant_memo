# 탕탕특공대 무기 업그레이드 체계 표

기준일: 2026-04-17

## 1. 전투 중 무기 업그레이드 체계

| 단계 | 조건 | 결과 |
| --- | --- | --- |
| 1성 획득 | 무기 또는 공격 스킬 선택 | 기본 공격 시작 |
| 2성 | 동일 스킬 재선택 | 공격력/범위/타수 등 강화 |
| 3성 | 동일 스킬 재선택 | 성능 강화 |
| 4성 | 동일 스킬 재선택 | 성능 강화 |
| 5성 | 동일 스킬 재선택 | 최대 일반 강화 달성 |
| 돌파(EVO) | 5성 + 대응 지원품 보유 | 상위 무기 형태로 진화 |

## 2. 전투 중 업그레이드 핵심 규칙

| 항목 | 내용 |
| --- | --- |
| 슬롯 제한 | 액티브 6칸, 패시브 6칸 |
| 장착 무기 시작점 | 장착한 인벤토리 무기가 기본 액티브 1칸 차지 |
| 돌파 방식 | 대부분 `5성 + 대응 지원품` |
| 예외 | 쌍생 광검은 2개 무기가 각자 5성이 되면 패시브 없이 병합 |
| 조합 무기 | 드론/지뢰 계열은 스킬 간 조합으로 상위 형태 생성 |

## 3. 장비 무기 합성 체계

| 합성 단계 | 일반 설명 |
| --- | --- |
| Normal | 기본 등급 |
| Good | 초급 상향 |
| Better | 중간 상향 |
| Excellent | 본격적인 고급 등급 시작 |
| Excellent +1 | 추가 강화 |
| Excellent +2 | 추가 강화 |
| Epic | 상위 등급 |
| Epic +1 | 추가 강화 |
| Epic +2 | 추가 강화 |
| Epic +3 | 추가 강화 |
| Legendary | 최종 전설 등급 |

## 4. 장비 합성 규칙 개요

| 규칙 | 설명 |
| --- | --- |
| 저등급 합성 | 같은 장비 여러 개를 소모해 상위 등급 제작 |
| 중간 구간 | 동일 희귀도 장비와 재료 장비를 활용해 `+1`, `+2`, `+3` 강화 |
| 전설 도달 | Epic 누적 합성 후 Legendary 달성 |
| 체감 효과 | 단순 공격력 증가뿐 아니라 등급 고유 스킬 해금 |

## 5. 무기 등급 고유 효과 예시

| 무기 | Better | Excellent | Epic | Legendary |
| --- | --- | --- | --- | --- |
| 빛을 쫓는 자 | 공격 간격 감소 | 일정 횟수마다 후방 베기 | 공격력 증가 | 검진 잔류 효과 |
| 파괴의 힘 | 투사체 범위 증가 | 블랙홀 종료 전 폭발 | 공격력 증가 | 보호막 파괴 시 폭발 |
| 혼돈의 검 | 스킬 피해 증가 | 영혼 족쇄 누적 | 공격력 증가 | 영혼 폭발 광역화 |

## 6. S급 무기 성장 체계

| 항목 | 내용 |
| --- | --- |
| 대표 무기 | 빛을 쫓는 자, 파괴의 힘, 혼돈의 검 |
| 획득 방식 | S급 공급 상자, 이벤트 |
| 특징 | 일반 장비 합성 라인과 병행하지만 자체 성능이 더 강한 편 |
| 성장 포인트 | 장비 등급, 전투 중 5성, EVO, 추가 강화 |

## 7. SS급 무기 쌍생 광검 체계

| 항목 | 내용 |
| --- | --- |
| 무기명 | Twin Lance - Starforged Havoc |
| 분류 | SS급 / Relic |
| 획득 방식 | Cosmic Cast |
| 해금 조건 | 첫 Legendary 장비 제작 후 Cosmic Cast 해금 |
| 제작 재화 | Eternal Core, Voidwaker Core, Base Tech Materials |
| 전투 특징 | 2개 무기처럼 동작하며 액티브 슬롯 2칸 사용 |
| 진화 방식 | Starforge와 Havoc이 각각 5성 도달 시 병합 |
| 병합 효과 | 1칸을 반환하고 상위 공격 패턴 활성화 |

## 8. Cosmic Cast 체계

| 항목 | 설명 |
| --- | --- |
| 역할 | SS급 장비를 제작하는 시스템 |
| 요구 조건 | 첫 전설 장비 제작 이후 해금 |
| 특징 | 일반 합성의 상위 레이어 |
| 무기 관련 대표 사례 | 쌍생 광검 제작 |

## 9. Astral Forge 체계

| 항목 | 설명 |
| --- | --- |
| 역할 | SS 또는 상위 장비에 추가 강화 레이어 제공 |
| Eternal 축 | 빛을 쫓는 자 계열 재료 사용 |
| Void 축 | 파괴의 힘 계열 재료 사용 |
| 효과 | 공격력 증가, 스킬 시작 레벨 보정, 추가 효과 부여 |
| 의미 | 장비 성장의 최종 단계권 |

## 10. 무기 업그레이드 흐름 요약표

| 단계 | 전투 중 | 전투 밖 |
| --- | --- | --- |
| 초반 | 무기 1성 -> 5성 | 장비 장착, 기본 합성 |
| 중반 | EVO 완성 | Excellent / Epic 달성 |
| 후반 | 조합 무기 최적화 | Legendary 달성 |
| 엔드게임 | SS 무기 병합 운용 | Cosmic Cast, Astral Forge |

| 구간 | 실제 중요한 요소 |
| --- | --- |
