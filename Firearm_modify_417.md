## 0. 한 줄 요약

탕탕의 **6계층 무기체계**(장착 무기 / S급 / SS급 / 일반 공격 스킬 / 무기 스킬 EVO / 조합 스킬)를 **출퇴근 테마로 변형**해서, **3단계 Phase**로 점진 도입. Phase 1은 v3.1 범위 내 실행 가능.

## 1. 탕탕 6계층 vs 우리 현재 상태 매핑

| # | 탕탕 계층 | 탕탕 예시 | 우리 현재 | 갭 |
|---|---|---|---|---|
| 1 | **장착 무기** (전투 전 인벤토리 선택) | 쿠나이, 리볼버, 샷건, 카타나, 야구배트 | 키보드 1개 고정 (선택 불가) | **시작 무기 선택 UI 미구현** |

| 4 | **전투 중 일반 공격 스킬** | 축구공, 드릴샷, 번개, 화염병, RPG, 드론, 두리안, 레이저, 수호자, 포스필드, 부메랑, 벽돌, 월영참, A/B/C드론, 모듈지뢰 (16종) | 우산, 텀블러, 스마트폰, 에너지바 (4종) | **스킬 풀 너무 좁음** (4/16) |

**진단**: 계층 4(일반 스킬)만 부분 구현. 계층 1(장착 무기) 개념 자체가 우리에게 없음. 6계층 구조를 그대로 쓸 필요는 없지만 **"장착 무기 vs 전투 중 스킬"** 구분은 설계 축으로 가져올 가치 있음.

## 2. 변형 원칙 5가지

**2.1 세션 길이에 맞게 축소**
탕탕 15분/8분 맵 기준 6계층 = 과잉. 우리 5분에서는:
- 장착 무기: **일반 5종** → 3종 (한 손 조작이라 큰 선택 피로 회피)
- 일반 공격 스킬: **16종** → 8~10종 (5분에 6슬롯 채울 정도)
- 슬롯: **액티브 6 / 패시브 6** → **액티브 4 / 패시브 4** (현재 3/4에서 소폭 확장)

**2.2 테마 통일: 사무실 도구**
탕탕 = 액션 영화 무기 (총/칼/수류탄). 우리 = 출근/직장. 전부 사무용품·식음료로 번역.

**2.3 EVO 조합은 "서사적 업그레이드"**
단순 수치 증가가 아닌 **"아 이거 말 되네"** 하는 변환. 예: 볼펜 + 고성능 자석 → 만년필 (귀족화).

**2.4 메타 성장은 v5.0+로 유예**
장비 합성, SS, Cosmic Cast, Astral Forge는 **라이브 서비스 단계 도달 후**. 프로토타입은 "5분 완결 세션"에 집중.

**2.5 조합 스킬 1~2개로 흥미 포인트 확보**
드론 조합 같은 "슬롯 절약 + 질적 변환"은 빌드 재미의 핵심. 복잡하지만 최소 1~2개는 Phase 1에 꼭 넣음.

## 3. 테마 매핑 — 사무실 아이템 변환

### 3.1 장착 무기 (계층 1, 시작 무기)

| 탕탕 | 특성 | 출퇴근 변형 | 특성 | 이유 |
|---|---|---|---|---|
| 쿠나이 | 추적 단일딜 | **🖊️ 볼펜** (Ballpoint Pen) | 자동 추적 찌르기 | 손에 잡히는 가장 일반적인 사무 도구 |
| 리볼버 | 수동 조준 단일딜 | **🔫 테이프 건** (Tape Gun) | 전방 단발 고화력 | 타격감 있는 투척 도구 |
| 샷건 | 전방 집중 | **🖨️ 프린터 용지** (Paper Spray) | 부채꼴 다발 발사 | "종이 뭉치가 쏟아지는" 느낌 |
| 카타나 | 근접 베기 | **📐 철자** (Steel Ruler) | 근접 전방 베기 | 칼처럼 생긴 사무용품 |
| 야구배트 | 넉백 근접 | **🥃 머그컵** (Mug) | 휘두르기 + 넉백 | 사무실에 늘 있는 둔기 |

→ **3종 선택** (MVP): **볼펜 / 철자 / 머그컵** (추적/근접/넉백 3축)

### 3.5 조합 스킬 (계층 6)

| 탕탕 | 우리 변형 | 공정 |
|---|---|---|
| A드론+B드론=파괴자 | 팩스봇A+팩스봇B=**프린터팜** | 2→1 슬롯, 광역 종이 폭격 |
| 지뢰+화염병=Inferno Bomb | 스타이로폼폭탄+에너지바=**발화 스타이로폼** | 장판+폭발 콤보 |
| 지뢰+번개=Thunderbolt Bomb | 스타이로폼폭탄+스마트폰=**번개 폭탄** | 낙뢰+폭발 콤보 |

### 3.6 패시브 연결 (계층 5)

현재 4종(커피/에너지드링크/도시락/교통카드)은 유지. Phase 2에 **고성능 자석** 대응 신규 패시브 추가:
- **🧲 손목밴드 자석** (코인 수집 범위 +) — 부메랑(사원증)/혼돈의검(서류함)의 EVO 키

## 5. Phase 1 구체 작업 (우선 실행)

### 5.1 주먹밥 (축구공 대응)

```js
// data.js
onigiri: {
  id: 'onigiri', name: '주먹밥', icon: '🍙',
  desc: '바닥을 튕기며 광역 타격',
  type: 'bounce',
  levels: [
    { dmg: 5,  cooldown: 1.5, speed: 320, bounces: 3, range: 260 },
    { dmg: 7,  cooldown: 1.4, speed: 340, bounces: 3, range: 280 },
    { dmg: 10, cooldown: 1.2, speed: 360, bounces: 4, range: 300 },
    { dmg: 12, cooldown: 1.1, speed: 380, bounces: 5, range: 320 },
    { dmg: 16, cooldown: 1.0, speed: 400, bounces: 6, range: 340 },
  ],
},
```

**구현**: 주먹밥 투사체가 가장 가까운 적 방향으로 발사 → 적 or 화면 경계에 닿으면 다음 가까운 적으로 튕김 → `bounces` 횟수 소진 시 소멸. 매 바운스마다 데미지 80% 감쇠 (탕탕 축구공 패턴).

### 5.2 스테이플러 (RPG 대응)

```js
stapler: {
  id: 'stapler', name: '스테이플러', icon: '📎',
  desc: '적에게 폭발 심지 발사',
  type: 'explode',
  levels: [
    { dmg: 12, cooldown: 2.0, speed: 420, range: 280, aoeRadius: 60 },
    { dmg: 16, cooldown: 1.8, speed: 440, range: 300, aoeRadius: 65 },
    { dmg: 22, cooldown: 1.6, speed: 460, range: 320, aoeRadius: 72 },
    { dmg: 28, cooldown: 1.4, speed: 480, range: 340, aoeRadius: 80 },
    { dmg: 40, cooldown: 1.2, speed: 500, range: 360, aoeRadius: 90 },
  ],
},
```

**구현**: 가장 먼 적에게 폭발 투사체 → 명중 or 사거리 끝에서 AOE 폭발 (스마트폰 AOE 로직 재활용).

### 5.3 조합 스킬 — 번개 폭탄 (스마트폰 + 스테이플러)

**판정 조건**: `스마트폰 Lv3+ 보유` AND `스테이플러 Lv3+ 보유` → 레벨업 시 **조합 카드** 등장 (가중치 8, EVO와 유사).

**효과**:
- 두 무기를 **병합** → 슬롯 2개 → **1개** 반환
- 새 무기 `lightning_bomb` 활성 (쿨다운 1.0s, AOE 반경 100px, 낙뢰 3회 + 폭발)

**코드 변경**: `LevelUp.js`의 `_generateOptions`에 조합 판정 블록 추가 (EVO 블록과 평행).

1. **Phase 1 착수 여부**: 주먹밥 + 스테이플러 + 번개 폭탄 조합 지금 구현?

3. **조합 스킬 우선순위**: 첫 조합을 스마트폰+스테이플러로 할지, 아니면 다른 조합 (예: 에너지바+스마트폰=화염 번개)?
4. **명칭 재검토**: "에너지바(화염병)"은 테마상 약함 — 🔥 "라면" (국물이 장판) 같은 대안 고려?
5. **Phase 2 선행 조건**: START 화면 현재 숨김 상태. 장착 무기 UI 추가 시 START 화면 재도입 필요 → OK?
